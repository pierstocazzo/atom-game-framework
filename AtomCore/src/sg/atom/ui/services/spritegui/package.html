A GUI library make up of Sprite elements for JME3.

Another native GUI libs but it has much more features.

P/s: I dont want to start a chaos that you always affraid :)

Features

  • GUI element is a composite
  • Layout is optional
  • Functional Reactive (ELM)
  • Caccased styles and scriptable styles (LESS)
  • Query (GQuery. DotQuery)
  • Advanced template (StringTemplate, MustacheMonkey)
